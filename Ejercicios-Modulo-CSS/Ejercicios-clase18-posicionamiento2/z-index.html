position z-index

El posicionamiento z-index basicamente indica sobre que capa queremos que este nuestro elemento, usamos numeros como capas, a mayor numero capa mas alta, si un elemnto tiene elemento 5 otro 3 y otro 8, visualmente se vera primero el 8 despues el 5 y despues el 3 en el caso que lleguen a cruzarse

En este caso la imagen azul tiene un position absolute pero tiene un position z-index 0 y el footer tiene un z-index -1, por lo que el footer queda detras de las imagen, si le colocamos 1 al footer quedara por delante de la imagen

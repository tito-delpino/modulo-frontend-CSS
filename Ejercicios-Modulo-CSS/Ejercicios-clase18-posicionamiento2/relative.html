position relative

Al posicionamiento relative es igual que el static, que es el que esta por defecto, pero del cual podemos hacer uso de direcciones para indicar hacia donde queremos mover el elemento, se interpreta desde los bordes de la caja contenedora, con esto si indicamos que el elemento tenga un desplazamiento left: 10px, el elemento se movera empujado desde la izquierda, se pueden utilizar valores negativos por los que si indicamos, left: -10px, el elemento se movera hacia la derecha, este tipo de posicionamiento no afecta a las cajas adyacentes ya que el posicionamiento relativo no afecta al comportamiento de las demas

Estas imagenes se colocan por defecto una al lado de la otra al ser elemento de linea

Si indicamos que la primera imagen se posicione con un top:100px, las demas imagenes no se moveran hacia donde estaba la primera, incluso se superpone la imagen1 sobre el section siguiente ya que este no cambia tampoco su posicion

Este section necesita un padding-top para poder verse las primeras palabras de este texto ya que sino estarian detras del cuadro superior 1, lo voy a pintar rojo oscuro para que se vea

position float

Esta imagen está flotando a la izquierda. Observa cómo el texto se ajusta a su alrededor, llenando el espacio disponible a la derecha. Este comportamiento era muy usado antes de que existiera Flexbox. Si este texto fuera más largo, seguiría fluyendo alrededor de la imagen.

Este texto está fuera del section. Si no “limpiamos” el float, puede subirse y colocarse junto a la imagen, rompiendo el flujo visual.

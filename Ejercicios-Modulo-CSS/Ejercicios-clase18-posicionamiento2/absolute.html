position absolute

El posicionamiento absoluto establece la posicion exacta del elemento, sale completamente del flujo normal del documento, significa que ya no ocupa espacio ni afecta a los demás elementos, el navegador lo coloca exactamente donde le digas, usando las propiedades top, right, bottom o left, se posiciona respecto al primer contenedor padre que tenga position: relative, absolute o fixed. Si ninguno de sus padres tiene posición establecida, se toma como referencia el html o la ventana completa.

Estas imagenes se colocan por defecto una al lado de la otra al ser elemento de linea

Si indicamos que la primera imagen se posicione con un top:100px y un right:100px, veremos como la imagen del perro que es la primera de este elemento contenedor section, se encuentra en la parte superior derecha del navegador ya que no tiene ningun ancestro que tenga posicionamiento relative, absolute o fixed.

Este section tiene posicionamiento static, el que viene por defecto como vemos la imagen se coloca a la izquierda y arriba del elemento padre, y hace que este texto se adapte a ella.

Este section sera el padre del elemento que queremos mover y tendra un posicionamiento relative para que el punto de partida para el calculo de la posicion del elemento img sea el mismo contenedor, como vemos la img tiene un top:130px y un left:100px, pero queda dentro del elemento, ya que esta asignado el position relative sobre el, por eso la imagen se queda dentro de este contenedor sobre el texto, vamos a rellenar el section para que sea mas grande

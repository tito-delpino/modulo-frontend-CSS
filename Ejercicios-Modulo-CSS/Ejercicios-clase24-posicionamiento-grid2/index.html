Display Grid

El sistema de elementos flexibles Flex es una mejora pero orientado en una sola dimension, con posicinamiento grid es posible crear una griya para darle tama√±o, posicion o colocacion, cambiando el nombre de sus clases, muchos conceptos de flex se toman parecidos en grid

Partes de un grid

  • Contenedor: es el elemento padre que contiene la cuadricula o rejilla
  • Item: es cada uno de los hijos que contiene el contenedor
  • Celda: cada uno de los cuadros de la cuadricula
  • Area: region o conjunto de celdas adyacentes de la cuadricula
  • Banda: linea horizontal o vertical de celdas de la cuadricula
  • Linea: separacion entre las celdas de la cuadricula

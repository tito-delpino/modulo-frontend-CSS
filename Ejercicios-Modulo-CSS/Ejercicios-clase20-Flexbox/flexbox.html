Repaso display

La propiedad display permite modificar el comportamiento de un elemento HTML,cambiandolo al que le indiquemos, inline, block o inline-block, a estos elementos podemos estanblecerle propiedades como ancho, alto, padding, etc..

Ejemplo de section con inline-block, un pading de 10 px y alto y ancho de 200px con un margin de 30px

flexbox

Con flexbox se abre una una gama de opciones para poder manejar los elementos del proyecto, hace que los elementos se adapten a nuestras necesidades mucho mas facil, funciona mediante los ejes X e Y, eje horizontal y eje vertical

Puede hacer referencia al contendor padre, usando la propiedad 'flex' sobre el container

flex-direction

El eje principal por defecto sera el horizontal pero podemos cambiarlo por el vertical, que seria el secundario, o viceversa, haremos referencia a ellos como flex-direction

*Como se ve en la imagen, por defecto viene configurado con flex-direction: row
